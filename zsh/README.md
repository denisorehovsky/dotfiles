
Zsh
===

> Why haven't you installed it already!?


### Installation

```bash
# Set up `zsh` (arch linux)
pacman -S zsh

# Install `oh-my-zsh`
sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

# Symlink `.zshrc`
ln -s ~/.dotfiles/Zsh/zshrc.symlink ~/.zshrc

```
